📋 Quick Reference

Copy-paste commands for common tasks.

⚡ Install & Run
# One-command install
curl -fsSL https://raw.githubusercontent.com/coinerd/q/main/scripts/install.sh | bash

# Interactive TUI
bin/q-tui

# Single-shot prompt
bin/q "explain this codebase"
🎯 Goal Mode
# Set a goal in TUI
/goal fix all failing tests in the auth module

# Clear a goal
/goal clear

# With custom evaluator
/goal --evaluator strict make it pass
🔌 Provider Config
# ~/.q/config.json
{
  "default-provider": "openai",
  "default-model": "gpt-5.4",
  "providers": {
    "openai": {
      "api-key-env": "OPENAI_API_KEY"
    }
  }
}
🧪 Testing
# Full test suite (parallel)
racket scripts/run-tests.rkt

# Fast suite (skip slow tests)
racket scripts/run-tests.rkt --suite fast

# Quick smoke test (~40s)
racket scripts/run-tests.rkt --suite smoke
