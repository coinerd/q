Why q?

Everything you need in a coding agent — nothing you don't.

🔒

Local-first

No cloud dependency. Your code stays on your machine. Run with local LLMs or cloud providers — you choose.

📝

Fully Auditable

Append-only session logs in JSONL format. Every action recorded, every decision traceable.

🔌

Provider-agnostic

Swap between OpenAI, Anthropic, Google Gemini, or local models without changing your workflow.

🧩

Extensible

Hook system for custom behavior at every lifecycle point. Build extensions that integrate seamlessly.

🖥️

Multiple Interfaces

CLI for scripts, TUI for interactive work, JSON mode for machines, RPC for integration, SDK for embedding.

🌿

Session Branching

Fork conversations, explore alternatives, merge back. Your agent history is a tree, not a line.

Get Started

Up and running in under two minutes.

1

Install Racket 8.10+

Download from https://racket-lang.org
2

Clone & Install

git clone https://github.com/coinerd/q.git
cd q
raco pkg install --auto
3

Configure Provider

# Create ~/.q/config.json with your provider
# See provider options below
4

Run

# Interactive TUI
bin/q-tui

# Or a single-shot prompt
bin/q "explain what this codebase does"

Provider Configuration

Choose your LLM — swap freely without changing your workflow.

☁️ OpenAI

gpt-5.4
{
  "default-provider": "openai",
  "default-model": "gpt-5.4",
  "providers": {
    "openai": {
      "base-url": "https://api.openai.com/v1",
      "api-key-env": "OPENAI_API_KEY"
    }
  }
}

☁️ Anthropic

claude-sonnet-4-6
{
  "default-provider": "anthropic",
  "default-model": "claude-sonnet-4-6",
  "providers": {
    "anthropic": {
      "base-url": "https://api.anthropic.com/v1",
      "api-key-env": "ANTHROPIC_API_KEY"
    }
  }
}

Security & Trust

Built with a security-first mindset from day one.

  • ✓

    Credential Protection

    API keys resolved via pluggable backend: env vars → OS keychain → file. Credentials stored with 0600 permissions. Never logged or printed.

  • ✓

    Append-Only Session Logs

    Every action recorded in JSONL format. The agent loop only appends — never rewrites history. Tamper-detectable by design.

  • ✓

    Shell Command Safety

    LLM-generated arguments are quoted via shell-quote to prevent injection. Sandboxing provides defense-in-depth with resource limits and timeouts.

  • ✓

    Small Trusted Core

    The core agent loop has minimal dependencies — no TUI, no CLI coupling. Security boundaries enforced at the architecture level.
