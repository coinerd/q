## First Steps

1. **Install q** — follow the [Installation Guide](../install.md)
2. **Verify** — run `racket main.rkt --version` and `racket main.rkt doctor`
3. **Configure** — set up API keys via environment variables or `~/.q/credentials.json`
4. **Run** — `racket main.rkt --tui` for the terminal UI, or `racket main.rkt "your prompt"` for single-shot
