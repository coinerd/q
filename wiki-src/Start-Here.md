## What q is

q is a local-first, extensible coding agent runtime with multiple interfaces and a layered architecture.

## First steps

1. Read the [README](https://github.com/coinerd/q/blob/main/README.md)
2. Follow the install instructions in the repo docs
3. Configure a provider
4. Run a first safe local session
5. Come back here for troubleshooting and architecture navigation
